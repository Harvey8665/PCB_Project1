# PCB_Project1
## 项目描述
个人PCB备份<br>
文件均为立创EDA文件，仅包含PCB图，不包含原理图<br>
## 使用方法
在立创EDA标准版、专业版、标准版（网页）、专业版（网页）导入即可<br>
## 注意事项
文件后面的版本指的是该文件所在项目的修改版本，与其他文件无关<br>
## 下载方式
部分文件点击文件名即可下载，如不可下载，可直接打包本工程下载<br>
